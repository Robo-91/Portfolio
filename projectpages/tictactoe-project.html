Tic Tac Toe

View Page Here

My implementation of the classic game. I made this web page as a learning tool, taken from the Odin Project Curriculum.

Factory Functions were used to create the player objects as an alternative to using Constructor Functions. Using factory functions, I was able to omit a fair amount of syntax (Not having to use the 'new' keyword, not setting up a constructor function).

Immediate Invoked Function Expressions (IIFE) were used to dynamically create the gameboard, keep track of the current player's turn, and determine who is the winner.
