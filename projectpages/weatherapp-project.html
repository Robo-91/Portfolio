Weather App

View Page Here

Context - Sudden weather changes can put a quick damper on your plans. I wanted to make an application that gives the user information regarding the weather in their area.

Action - I used Javascript's asynchronous features (async/await) to fetch information from an open source API. The goal for the interface was to make sure it was simple for the user.

Results - A simple weather application that displays the current weather in a given city. All the user needs to do is type in the city.
